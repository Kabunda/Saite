🧠 Как это работает
Пользователь открывает страницу, видит экран авторизации.

После входа через Google (или Яндекс) onAuthStateChanged переключает на главное меню.

Нажатие «Играть» запускает поиск комнаты в lobby.js – либо присоединяется к существующей, либо создаёт новую.

Когда в комнате двое, статус меняется на playing, начинается game.js.

Игроки нажимают «Сделать ход», счет обновляется в Realtime Database, соперник видит изменения мгновенно.

При достижении 5 очков игра завершается, статистика пишется в профиль, показывается результат.

Кнопка «В главное меню» возвращает в меню и очищает слушатели.

Весь код рассчитан на модульную структуру, легко модифицируется под реальную игру. Firebase обеспечивает синхронизацию в реальном времени без серверного бэкенда, а GitHub Pages – бесплатный хостинг статики.
