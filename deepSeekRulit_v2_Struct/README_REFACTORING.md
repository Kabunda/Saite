### 2. Создание модульной архитектуры
Создана следующая структура каталогов:

```
js/
├── app.js                          # Точка входа приложения
├── screens/
│   └── screenManager.js           # Управление видимостью экранов
├── game/
│   ├── gameLogic.js               # Логика игры (состояние, проверка ответов)
│   └── taskGenerator.js           # Генератор вопросов
├── data/
│   ├── storage.js                 # Единый интерфейс Firebase + LocalStorage
│   └── leaderboard.js             # Логика таблицы лидеров
├── ui/
│   ├── progressBar.js             # Управление прогресс-баром
│   ├── timer.js                   # Таймер игры
│   ├── keypad.js                  # Экранная и физическая клавиатура
│   └── modal.js                   # Универсальный менеджер модальных окон
├── settings/
│   └── settingsManager.js         # Управление настройками
├── media/
│   ├── sound.js                   # Звуковые эффекты
│   └── vibration.js               # Вибрация
└── utils/
    ├── dom.js                     # DOM хелперы (escapeHtml, showToast и др.)
    └── constants.js               # Общие константы приложения
```

### 3. Реализованные модули

#### Основные модули:
1. **screenManager.js** - Единая точка управления видимостью экранов
   - Методы show(), hideAll(), getScreenElement()
   - Поддержка событий screenchange
   - Автоматическое обнаружение экранов в DOM

2. **gameLogic.js** - Чистая логика игры (без зависимостей от DOM)
   - Класс GameLogic с методами checkAnswer(), nextQuestion(), getStats()
   - Управление состоянием игры (очки, серии, ошибки, время)
   - Рассчет штрафного времени

3. **taskGenerator.js** - Генератор вопросов
   - Функции buildQuestionList(), buildUniqueQuestionList()
   - Поддержка балансировки вопросов по сложности
   - Экспорт констант для обратной совместимости

4. **storage.js** - Унифицированный интерфейс хранения
   - Работа с Firebase (при доступности)
   - Локальное хранение в LocalStorage
   - Офлайн-синхронизация результатов
   - Управление настройками игрока

5. **leaderboard.js** - Логика таблицы лидеров
   - Формирование HTML для отображения
   - Сортировка с учетом штрафного времени
   - Поддержка личных рекордов

#### UI компоненты:
6. **progressBar.js** - Интерактивный прогресс-бар
   - Визуализация прогресса игры
   - Анимации правильных/неправильных ответов
   - Поддержка ARIA атрибутов

7. **timer.js** - Таймер игры
   - Точный отсчет времени
   - Форматирование MM:SS
   - События onTick

8. **keypad.js** - Управление вводом
   - Обработка экранной клавиатуры
   - Поддержка физической клавиатуры
   - Визуальная обратная связь

9. **modal.js** - Универсальные модальные окна
   - Управление открытием/закрытием
   - Поддержка оверлея
   - Обработка клавиши Escape

#### Настройки и медиа:
10. **settingsManager.js** - Централизованное управление настройками
    - Загрузка/сохранение настроек
    - События при изменении настроек
    - Управление темой приложения

11. **sound.js** - Звуковые эффекты
    - Воспроизведение тонов через Web Audio API
    - Управление состоянием (вкл/выкл)
    - Предопределенные звуки для событий игры

12. **vibration.js** - Вибрация
    - Проверка поддержки браузером
    - Паттерны вибрации для разных событий
    - Интеграция с настройками

#### Утилиты:
13. **dom.js** - DOM хелперы
    - escapeHtml() для безопасного отображения
    - showToast() для уведомлений
    - createElement() для создания элементов

14. **constants.js** - Общие константы
    - Настройки игры, ключи LocalStorage
    - Идентификаторы экранов и модальных окон
    - Частоты звуков и паттерны вибрации

#### Точка входа:
15. **app.js** - Главный модуль приложения
    - Инициализация всех менеджеров
    - Координация работы модулей
    - Обработка глобальных событий
    - Управление состоянием приложения
